Athena's Memory

$59.99

The game world is set in Athena’s android memory system. The 3D environment started off with simple geometric shapes to represent the current memory Athena is holding in her memory system (plain). As the player proceed through the stages of the game, architecture and details will be slowly added into the environment to indicate the player’s process and presents the current stage of the story.

Release Date: April 1, 2018

Adventure Of Blobby

$9.99

Adventure of Blobby is a 2D platformer created in Unity on the PC and mobile platform.The game is about a lost blob trying its hardest to navigate through the dangerous forest, solving multiple puzzles and defeat spooky monsters to find and reach its family. The game focuses on the mechanics which allow Blobby to transform into different shapes to overcome the obstacles in the game.

Release Date: March 20, 2017

Another Me

$19.99

Another Me is a 2D platformer game set in a post-apocalypse world where the human race is extinct, There are only the android robots that was created by the human left on the planet. The main protagonist Serena set out on a journey to find her true self and her background.

Release Date: N/A

Adrift

$9.99

Adrift Vr is an immersive VR experience designed through Unity. This Vr experience is created to empathize on the connection between children and parents. Adrift Vr allow the audience the be fully immersed into the children’s dream through the first person simulation.

Release Date: July 10, 2017
